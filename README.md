# Quiz de Telefonia — Website

Este projeto é um site React + Vite que roda no navegador.

Para rodar localmente a partir da pasta `src`:

```powershell
Set-Location 'C:\Users\NEXTSTAGE\Desktop\ERICK\src'
npm install
npm run dev
```

Scripts disponíveis:
- `npm run dev` — inicia o site em desenvolvimento com Vite
- `npm run build` — gera a versão de produção em `dist`
- `npm run preview` — serve o build de produção localmente para revisão
